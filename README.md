--------frontend & BACKEND------
1.clone repo from gitHub (Frontend and Backend)
2.run npm install 
3.run npm run dev 

4.for backend i have used SpringBoot 
5.to run backend you need springWorksace whre you can run 
6.you need to crate one local database to work after that proceed with next
7.after getting into project folder in backend go to 
    project -> src -> main -> resources -> application.properties 
            spring.datasource.username={DATABASE USERNAE}
            spring.datasource.url=jdbc:{URL}  ---       (postgresql://localhost:5432/E-Commerce)
            spring.datasource.password={PASSWORD}

    Set all this first before runnuing backend 
8. To Run spring RightClick on Project -> Run AS -> Java Application
